Reglas "Vóley Shot" 🍸

  • 🤝 Coordinación: Cada equipo debe coordinarse para traer su botella o trago favorito. ¡Que nadie juegue con la garganta seca!
  • 🥃 Pena máxima: El equipo que pierda un set se toma un vaso (shot). ¡El castigo del perdedor es brindar!
  • 🏐 Reglamento: Se respetarán las reglas básicas del vóley y se cobrarán las faltas evidentes (dobles, retención) para asegurar un juego limpio.
  • 🚫 Malla sagrada: Se jugará sin tocar la malla. Si la tocas, ¡te toca tomar un shot adicional de penalización!
